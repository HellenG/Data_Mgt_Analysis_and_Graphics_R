Preface

Welcome to “Fundamentals of Data Analysis using R”, this is the third and core book in our learning data analysis and graphics tutorial series.

In this book we aim to get a good foundation on data analysis; that is, the fundamental concepts and methods in Statistics. In all our chapters, we will strive to understand the reasoning behind the calculations including assumptions made rather than memorizing formulars. In so doing we get the ability to make sound interpretation of results.

In this book, we will begin by learning how to describe data before learning how to make inferences.
